Crack Airtel Card

A few days ago I was asked to crack a card. So today I thought to write a post for this method.

Prerequisites

  • A linux machine
  • log file
  • hashcat
  • Some common sense

Getting the hashes

Open log file and search for CW and decrypted CW. Remove the leading zeros and copy the last 16 characters for first appearing CW’s. Do same with decrypted CW. Enjoin them with a : so it will look like this:

1234567890123456:1234567890123456

Installing required software

hashcat is available in most linux distributions but installing it from your package manager isn’t preferred. Grab the latest archive and install it. Assuming you have a Ubuntu installation. Let’s install the required packages using Ubuntu package manager first:

$ sudo apt update && sudo apt upgrade
$ sudo apt install -y build-essential unzip unrar tmux curl p7zip ocl-icd-libopencl1 opencl-headers clinfo nvidia-driver-390 nvidia-cuda-toolkit
$ nvidia-smi
$ sudo reboot -f
$ sudo add-apt-repository ppa:graphics-drivers/ppa
$ sudo apt update
$ sudo apt install -y mesa-common-dev freeglut3-dev
$ sudo reboot -f

Let’s fetch the latest hashcat from hashcat’s website.

$ curl -o hashcat-6.1.1.7z https://hashcat.net/files/hashcat-6.1.1.7z
$ 7zr x hashcat-6.1.1.7z
$ cd hashcat-6.1.1
$ sudo ln -s hashcat64.bin /usr/bin/hashcat
$ sudo cp -R OpenCL/ /usr/bin/
$ sudo cp hashcat.hcstat2 /usr/bin/
$ sudo cp hashcat.hctune /usr/bin/

Start cracking

Now we have installed required software. Let’s start the real work.

Start a tmux session and issue following command. Replace CW:DCW with what you have copied above.

$ hashcat -m 14000 CW:DCW -a 3 -1 /usr/share/hashcat/charsets/DES_full.charset -o key.txt --hex-charset ?1?1?1?1?1?1?1?1 -w 4 --outfile-format 5

Once command is completed, you will get the cracked key in key.txt.

Happy Cracking

Congrats!. You just have cracked the card. If this tutorial helped you, don’t forget to share and leave your thoughts in comments section.
